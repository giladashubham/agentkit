# Security Policy

## Supported versions

AgentKit is currently pre-1.0. Security fixes are applied to the latest released version.

## Reporting a vulnerability

Please do not open a public issue for suspected vulnerabilities.

Report security concerns by emailing the maintainer listed in `pyproject.toml`, or by using GitHub private vulnerability reporting if it is enabled for the repository.

Include:

- affected version or commit
- reproduction steps
- expected impact
- any suggested fix, if available
